Project Overview
This project is an end-to-end Business Intelligence solution built using Power BI to analyze sales performance, product profitability, and retailer behavior. The dashboard is designed to support executive decision-making by providing interactive insights across time, regions, products, and retailers.

Business Objectives
1. Monitor overall sales, profit, and growth trends
2. Identify top-performing products, retailers, and regions
3. Analyze profitability using contribution margin and profit %
4. Evaluate retailer value using Customer Lifetime Value (CLV)
5. Enable data-driven strategic and operational decisions
